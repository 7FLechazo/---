# ---下面是本项目的CSS样式文件介绍<br />
1.base.css样式文件是页面初始化<br />
2.common.css样式文件是项目页面的公共样式 如页面头部和底部模块 版权模块的样式 可以根据需要link到html文件中即可<br />
3.index.css样式是页面主体部分样式<br />
4.list.css是部分产品样式<br />
5.registe.css样式是用户注册页面样式<br />
<br />
# ---下面是本项目的HTML结构文件介绍<br />
